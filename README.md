# AI Predictive Maintenance System

## Project Overview
This project is an Arduino-based predictive maintenance system for agricultural machinery. It monitors temperature using a sensor and gives a buzzer alert when the temperature goes above the safe limit.

## Components Used
- Arduino UNO
- LM35 Temperature Sensor
- Buzzer
- Jumper Wires
- Breadboard

## Working Principle
The temperature sensor reads machine temperature. Arduino processes the sensor value. If the temperature is above 70°C, the buzzer turns ON to indicate overheating.

## Applications
- Agricultural machinery monitoring
- Motor overheating alert
- Farm equipment safety system
- Preventive maintenance system

## Arduino Code
The Arduino code is available in `arduino_code.ino`.

## Output
When temperature is normal, buzzer is OFF.  
When temperature is above 70°C, buzzer is ON.
